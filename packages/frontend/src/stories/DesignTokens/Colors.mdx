# Color Tokens

The x402Arcade color system is built on a **Retro Arcade / Neon** theme, featuring vibrant neon accents against deep, dark backgrounds.

## Philosophy

- **Dark canvas**: Deep purple-tinted blacks create the arcade atmosphere
- **Neon highlights**: Cyan and magenta bring the classic arcade glow
- **High contrast**: Light text on dark backgrounds for readability
- **Semantic clarity**: Consistent colors for success, warning, and error states

---

## Background Colors

These dark purple backgrounds create the deep space/arcade atmosphere.

| Token | Hex | Purpose |
|-------|-----|---------|
| `backgrounds.primary` | `#0a0a0f` | Darkest layer, page background |
| `backgrounds.secondary` | `#12121a` | Section backgrounds |
| `backgrounds.tertiary` | `#1a1a2e` | Cards, elevated areas |
| `backgrounds.surface` | `#16162a` | Modals, dropdowns |

### Usage

```tsx
import { backgrounds } from '@/styles/tokens/colors';

<div style={{ backgroundColor: backgrounds.primary }}>
  Darkest background
</div>

// With Tailwind
<div className="bg-bg-primary">
  Using Tailwind class
</div>
```

## Glow Colors

Semi-transparent colors for authentic neon glow effects.

| Token | Value | Use Case |
|-------|-------|----------|
| `glows.cyan` | `rgba(0,255,255,0.3)` | Subtle cyan glow |
| `glows.cyanStrong` | `rgba(0,255,255,0.5)` | Intense cyan glow |
| `glows.magenta` | `rgba(255,0,255,0.3)` | Subtle magenta glow |
| `glows.success` | `rgba(0,255,136,0.3)` | Success glow |
| `glows.warning` | `rgba(255,170,0,0.3)` | Warning glow |
| `glows.error` | `rgba(255,51,102,0.3)` | Error glow |

### Usage

```tsx
import { glows } from '@/styles/tokens/colors';

<button style={{
  boxShadow: `0 0 20px ${glows.cyanStrong}`
}}>
  Glowing Button
</button>
```

## Best Practices

### Do's

- Use `backgrounds.primary` for the main page background
- Use `primary.DEFAULT` for primary actions and links
- Use `semantic.*` colors for feedback states
- Apply `glows.*` for hover and focus effects

### Don'ts

- Don't use raw hex values; always use tokens
- Don't use colors outside the design system
- Don't combine multiple glow colors (can be overwhelming)
- Don't use light backgrounds (breaks the arcade theme)
